Quickstart

Include from dist/

    ...
    <link rel="stylesheet" href="hint.css">
    <script src="hint.js"></script>
    ...

Add your class and data-hint attribution to node

    ...
    <span class="some_class" data-hint="hint_text">some term</span>
    ...

Initialize hints

    
    document.getElementsByClassName("some_class").hint();

Options

pin: false | true

A little triangle pointer to initial node

animate: false | true

Fade animation

trigger: "mouseover" | "click"

Event to listen

timer: Integer (ms)

Hides a hint automatically in milliseconds 0 - never hide

delay: Integer (ms)

Time delay before showing (sometimes usefull when trigger is mouseover)

vertical: "auto" | "top" | "middle" | "bottom"

Set vertical position of hint. Middle position has issues

closeBy: ["mouseOut" | "scroll" | "resize" | "clickOutside" | "button"]

Set array of triggers to close the hint. Supported parameters:

  • mouseOut
  • resize
  • scroll
  • clickOutside
  • button (exit button inside hint)

Note: hint opened by mouseover will be closed by mouseout automatically

theme: "light" | "dark"

Hint styling


text: String

Allow to force your text or HTML instead of data-hint attribute.


stickiness: Integer (px)

Line break support

Lifehacks

  • Hint.js allow to set hint text by text option instead of data-hint attribute
  • Hint.js allow to set link to another node with text option or data-hint attribute by ID selector 
        ...
    <span class="some_class" data-hint="#my_hint">some term</span> 
    <div id="#my_hint">Hint content</div>
    ...
  • Hint.js supports line breaked text, but this feature may work incorrectly. Set a special stickiness option, that equals line-height property of text.
    For example line-height is 24px: 
        
    document.getElementsByClassName("some_class").hint({
        ...
        stickiness: 24,
        ...
    });
  • Set different options to different nodes: init Hint.js by different class name with differents options list
  • Edit CSS styles manually by .html-hint .html-hint-content and .html-hint .html-hint-pin selector
